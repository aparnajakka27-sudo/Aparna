# May 2026 Habit Tracker: Getting 1% Better Every Day

A minimalist, high-fidelity web application designed for tracking habits and sleep patterns during May 2026. Inspired by the philosophy of continuous improvement, this tracker helps you visualize your progress and stay consistent.

![Habit Tracker Preview](https://via.placeholder.com/1200x600/000000/FFFFFF?text=May+2026+Habit+Tracker+Preview)

## ✨ Features

- **31-Day Interactive Grid**: Track up to 12 habits with a simple tap-to-cycle system (Neutral, Done ✓, Missed ✕).
- **Sleep Monitoring**: Log daily sleep hours (5hrs–9hrs) in a dedicated visual grid.
- **Data Vault**: An integrated history log that records every manual save, providing a "Monthly Completion Rate" and a timeline of your progress.
- **Sticky Layout**: Optimized for mobile with a sticky first column, ensuring habit names are always visible while scrolling through the month.
- **Local Persistence**: All data is automatically saved to `localStorage`, so your progress is never lost on refresh.
- **Premium B&W Aesthetic**: A clean, distraction-free design using the Inter typeface and SVG-backed iconography.

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/aparnajakka27-sudo/Aparna.git
   ```
2. Open `index.html` in your browser.
3. (Optional) Run using a local server:
   ```bash
   npx serve .
   ```

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with Grid, Flexbox, and Custom Properties.
- **JavaScript**: Pure ES6 logic for state management and DOM manipulation.

## 📝 How to Use
1. **Renaming Habits**: Click on any habit label (e.g., "Habit 1") to type your own habit name.
2. **Marking Progress**: Click a cell to cycle through status (Checkmark for success, Cross for a missed day).
3. **Saving**: Click **"Save Progress"** to commit your daily data and view your summary.
4. **Analysis**: Use the **"View Data"** button to open the Data Vault and see your monthly success percentage.
